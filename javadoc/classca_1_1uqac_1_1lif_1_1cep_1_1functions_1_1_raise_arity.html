ca.uqac.lif.cep.functions.RaiseArity Class Reference

Detailed Description

A Function that raises the arity of another function.

Given an m:n function f, an instance of RaiseArity r makes f behave like an m':n function, with m' > m. The extra arguments given to r are simply ignored.
