Detailed Description

Delegates all calls of the Function class to an internal object instantiated by a method.

The main purpose of this class is to allow one to create a class out of a FunctionTree instance created programmatically. The code creating the FunctionTree is written in the getFunction() method (which, as a matter of fact, can return any other Function object).
