Detailed Description

The "next" status of a Pullable object.

Indicates whether a new output event is available (i.e. can be "pulled").

  • YES indicates that a new event can be pulled right now, using either pullSoft() or pull()
  • NO indicates that no event is available, and will ever be. Receiving NO generally indicates that the end of the (output) trace has been reached
  • MAYBE indicates that no event is available, but that keeping on pulling may produce an event in the future. This value is only returned by hasNextSoft().
