ca.uqac.lif.cep.Pullable.PullableException Class Reference

Detailed Description

A runtime exception indicating that something went wrong when attempting to check if a next event exists.

This happens, for example, if one of a processor's inputs it not connected to anything. Rather than throwing a NullPointerException, the issue will be wrapped within a PullableException with a better error message.
