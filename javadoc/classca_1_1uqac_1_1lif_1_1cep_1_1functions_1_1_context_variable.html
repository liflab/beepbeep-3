ca.uqac.lif.cep.functions.ContextVariable Class Reference

Detailed Description

Placeholder for the value of a context element.

A ContextVariable can be given as an argument to a FunctionTree.

outputsThe outputs of the function. The size of the array should be equal to the function's declared output arity.
