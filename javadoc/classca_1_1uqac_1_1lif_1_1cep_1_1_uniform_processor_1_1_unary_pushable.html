ca.uqac.lif.cep.UniformProcessor.UnaryPushable Class Reference

Detailed Description

A special type of Pushable for uniform processors with an input and output arity of exactly 1.

In such a case, the pushable object can operate in a much simpler way than the generic SynchronousProcessor.InputPushable defined by SynchronousProcessor, by foregoing the use of input and output queues completely.

Simple experiments with a Passthrough processor have shown a speed boost of about 3× compared to SynchronousProcessor.InputPushable.
