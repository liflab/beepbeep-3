ca.uqac.lif.cep.GroupProcessor.OutputCallAfterConnect Class Reference

Detailed Description

A CallAfterConnect object that can be used to connect the underlying processor of a GroupProcessor, and then to collect all processors that are part of the group.

Currently the only use of this class is to be returned by the out(Processor) method. (And in turn, the only real use of this method is in Groovy scripts to save a few keystrokes when creating a group.)
