ca.uqac.lif.cep.util.Bags.FilterElements Class Reference

Detailed Description

Gets all the elements of the collection that satisfy some condition.

This condition is specified as an unary function that is successively applied to each element of the collection; if the function returns true, the element is added to the output result, otherwise it is discarded.

This function preserves the type of the input. That is, if the input is a set, it will return a set; if the input is a list, it will return a list.
