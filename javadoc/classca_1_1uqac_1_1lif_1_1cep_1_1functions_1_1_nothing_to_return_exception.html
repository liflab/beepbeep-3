ca.uqac.lif.cep.functions.NothingToReturnException Class Reference

Detailed Description

Exception thrown when a function cannot produce a return value.

This is a "generic" type of exception; for example, if the function cannot return a value because one of its arguments is of the wrong type, it should rather throw an InvalidArgumentException.
